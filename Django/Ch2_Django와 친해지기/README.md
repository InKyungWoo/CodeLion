  ## 1. 딕셔너리
  
  - 데이터들을 `대응` 시켜주는 자료형
  - 탐색의 기준인 `키워드` ==== `찾고자 하는 값`을 대응시킴
  - 즉, `key` ==== `value`
  - 🌟 key값은 중복되어서도, 변해서도 안된다!

  💡 딕셔너리를 쓰는 경우 : 2 x N 표를 통해 데이터를 표현하고 싶을 때
